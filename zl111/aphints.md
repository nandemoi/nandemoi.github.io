### Python 輸出以空格相隔的多個數字

在 ```print (...)``` 的括弧中把要輸出的數字<span style="color:red">或變數</span>依序以逗號分開，例如：

```python
a = 1
c = 3
print (a, 2, c)
```

就會在螢幕上輸出

```
1 2 3
```

### Python 輸出文字

把你要的文字放在 ```"..."``` 裡，然後整個連 "" 放在 ```print (...)``` 的括弧裡就可以了，例如：

```python
print ("hello world")
```

就會在螢幕上輸出

```
hello world
```
