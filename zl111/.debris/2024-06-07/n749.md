# ZeroJudge《n749 最大和 (輸出簡化版)》解

```python
n, m = map (int, input().split())

maxes = []
for _ in range (n):
    nums = list (map (int, input().split())) # 讀入每列 (組) 數字 (組成一個 list)
    maxes.append (max (nums)) # append max of nums to maxes

s = sum (maxes) # sum of maxes
print (s)

for mx in maxes:
    if s % mx == 0:
        print (mx)
```
