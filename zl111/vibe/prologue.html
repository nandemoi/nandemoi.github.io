Vibe Coding 學 Python

Vibe Coding 是什麼？

Vibe Coding (共感編程) 是 AI 應用最新的發展之一。想像你有一位聰明又可靠的助手：你用你平常和朋友說話的方式告訴它一個模糊的點子，它會回覆一段程式、設計草圖或一個互動範例。你再修正你的敘述，它再改一次，直到成果感覺對了。

這就是 Vibe Coding 的精髓：以語氣、意圖與反覆修正來與 AI 共創應用程式。

Vibe Coding 強調共感 (Vibe)、互動 (Interaction)、與創造性程式邏輯 (Creative Logic)。

課程設計

對你初學 Python 和程式設計而言，嘗試 Vibe Coding 可能會有兩個挑戰：

  1. 中文對話：Vibe Coding 運用大語言模型 AI 需要經過大量資料的訓練，現有平台使用的模型還是以英文訓練為主，使用中文對話可能會需要多一點耐心。
  2. AI 生成的程式常常會比較複雜。Vibe Coding 要注意的一點是我們不希望 AI 幻想出一些沒有要求他做的事，所以還是要知道它生成的程式實際上會做什麼。太過複雜的程式對你初學想去看懂是一個學習上的負擔。

基於這兩個潛在的困難，這套課程設計上旨在以一個模擬體驗的方式，示範你能夠如何以 Vibe Coding 的方式創造程式，同時在過程中，認識與學習 Python 程式的基礎邏輯與運作原則。

透過模擬與互動練習，你會體會如何將想法轉化為可被電腦理解的邏輯陳述，逐步了解程式邏輯如何反映人的思考，學習以結構化的方式清楚表達意圖，培養將問題拆解、表達邏輯關係、與檢驗結果的能力。這些都是與 AI 協作與創意開發的基礎。
